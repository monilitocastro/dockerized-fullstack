Get Started
-----------

To build all, run
<code>./build-and-run-all.sh</code>
from project root

Then checkout `http://localhost:3000` in your local browser.

Server is served on port `9090`

Clean up
--------

To clean up, run
<code>./uninstall-all.sh</code>
from project root

Requirements
------------

- Bash
- Docker
